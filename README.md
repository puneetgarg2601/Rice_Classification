# Rice Type Classification
A PyTorch-based neural network to classify rice types.
## Dataset
- Source: Kaggle (Rice Type Classification)
- Features: 10 (e.g., length, width)
## How to Run
1. Clone: `git clone <repo-url>`
2. Install: `pip install torch pandas numpy matplotlib`
3. Run: `jupyter notebook Rice_Classification.ipynb`
## Results
- Test Accuracy: 98.7%
- See `Rice_Classification.ipynb` for details.
